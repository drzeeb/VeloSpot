Built for Everyday City Cycling

VeloSpot combines official Trier geoportal data, local SQLite caching, and a practical Android UI to help cyclists find the right bike parking spot in seconds.

  • Favorite spots highlighted with red map markers
  • Dedicated favorites sheet with direct navigation
  • Location recenter button and current-position marker
  • Dark mode toggle directly in the map menu
  • 8-language picker with persistent flag-based selection
  • Detailed parking metadata including capacity and coverage
  • Room-powered local caching for fast reloads
