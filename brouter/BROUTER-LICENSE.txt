This file documents the license of BRouter, which the :brouter Gradle module
compiles from source (no binary is committed to this repository).

Source code: https://github.com/abrensch/brouter
Version:     1.7.9  (pinned via the `brouter-upstream` git submodule)
License:     MIT (as stated in the BRouter repository)

The :brouter module compiles only the on-device routing modules:
  - brouter-codec       (btools.codec)
  - brouter-expressions (btools.expressions)
  - brouter-core        (btools.router)
  - brouter-mapaccess   (btools.mapaccess)
  - brouter-util        (btools.util)
The server / map-creation modules (btools.server, btools.mapcreator) and their
protobuf / osmosis dependencies are intentionally excluded — they are only used
to build map data, not to calculate routes on-device. All compiled modules are
MIT-licensed and contain no proprietary dependencies.

Build provenance:
  - Every build (local, CI and F-Droid) compiles the exact same source pinned
    by the submodule commit, so the result is reproducible.
